#PigeonHole Sort

*Implementaion of Pigeonhole sort in C# language,console based program using VisualStudio(IDE). 
*For academic purpose only.
*pigeonhole sorting algorithm is used to sort array elements.
 In O(n+N)time complexity.
*Given JPG file contains description of project.
*All files related to project are included.

*Platform:
-Visual Studio (IDE)

*How to run?
-Enter number of element to be entered.
-Enter unsorted elements in array and get desired ouput of sorted array.

*Works for?
-All negative and positive integer values.
